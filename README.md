### 使用方式 
- [前往Release下载最新包](https://github.com/fallingrust/XMagnetSearch/releases)
- [安装Dotnet Desktop Runtime](https://dotnet.microsoft.com/zh-cn/download/dotnet/8.0)
- 输入资源名称，回车
- 双击复制链接至剪切板
